# dashdub

Convert speech to text with Dash &amp; Python

Clone and run with `python3 dashdub.py`

#### Dependencies
- `ffmpeg` for [pydub](https://github.com/jiaaro/pydub)
- `Sphinx` for [speech_recognition](https://github.com/Uberi/speech_recognition)
